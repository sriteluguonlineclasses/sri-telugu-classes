# Sri Telugu Classes — Marketing & Leads

This folder holds your lead-generation toolkit. None of these files are linked from the
website, so they stay private to the repo (don't paste real student data into a public repo —
keep the *filled-in* tracker in Google Sheets instead, see below).

## Files

| File | What it's for |
|------|---------------|
| `leads-tracker.csv` | A simple CRM template. One row per lead. |
| `outreach-kit.md` | Ready-to-post Facebook content, the free-trial offer, and follow-up email + WhatsApp templates. |
| `marketplace-listings.md` | Profile copy to paste into UrbanPro, Superprof, and Preply. |

## How the leads reach you

Your website's "Get Free Trial" form (on `index.html`) now collects **name, email,
WhatsApp/phone, location, who's learning, age, and course of interest**. On submit it emails
all of that to **sriteluguonlineclasses@gmail.com** via Web3Forms. When a lead email lands,
copy the details into the tracker below.

## Using the lead tracker

1. Go to [sheets.google.com](https://sheets.google.com) → **Blank** → **File ▸ Import ▸ Upload**
   and pick `leads-tracker.csv`. (Or just open it in Excel.)
2. Delete the EXAMPLE row.
3. Add one row per new lead. Update the **Status** as they move through the funnel.

### Status values (the funnel)

`New` → `Contacted` → `Trial Scheduled` → `Trial Done` → `Enrolled`
(dead ends: `Not Interested`, `Lost`)

### Column notes

- **Timezone** — critical for NRI families; it decides which batch/slot you offer.
- **Source** — where the lead came from (which FB group, marketplace, referral). Tells you
  what's actually working so you double down on it.
- **Owner** — which teacher (Jyothi / Vijaya) is following up, so nobody falls through.
- **Follow-up Date** — sort by this column every morning; anyone overdue gets a nudge.

### A good daily rhythm

- **Morning:** sort by Follow-up Date, message everyone due today.
- **New lead:** reply within a few hours (speed massively raises conversion), set status to
  `Contacted`, and book the free 60-min trial.
- **After a trial:** mark `Trial Done`, then follow up within 24h with the "post-trial" message
  in `outreach-kit.md`.
